## 📖 项目概述

### 1.1 项目定位
EasyWechatWeb 是 EasyProject 项目的后端 API 服务，提供完整的 RESTful API 接口，支持 PC 管理后台、移动端 H5、微信小程序三端应用。

### 1.2 核心功能
- 用户认证与授权 (JWT)
- 基础数据管理 (用户/角色/菜单/字典)
- 业务工作流管理
- ETL 数据集成
- 报表管理
- 大屏设计
- 鲜花商城业务

## 🔧 技术实现

### 4.1 核心技术
- **框架**: ASP.NET Core 8.0
- **ORM**: SqlSugar 5.1
- **认证**: JWT Token
- **依赖注入**: Autofac
- **日志**: Serilog + 数据库日志
- **文档**: Swagger/OpenAPI

### 4.2 数据库设计
- **数据库**: MySQL 8.0
- **连接池**: 主从分离
- **表数量**: ~40 张
- **字符集**: utf8mb4

### 4.3 API 规范
```csharp
// 统一响应格式
public class ApiResponse<T>
{
    public int Code { get; set; } = 200;
    public string Message { get; set; } = "success";
    public T? Data { get; set; }
}

// 分页请求
public class PageRequest
{
    public int PageIndex { get; set; } = 1;
    public int PageSize { get; set; } = 10;
}

// 分页响应
public class PageResponse<T>
{
    public List<T> List { get; set; }
    public int Total { get; set; }
}
```
