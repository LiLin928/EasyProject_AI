## 3. ORM 数据库访问

### SqlSugar

| 包名 | 版本 | 用途 |
|------|------|------|
| SqlSugarCore | 5.1.4.197 | ORM 框架 |
| MySql.Data | 8.3.0 | MySQL 驱动 |

**核心文件：** `SqlSugar/SqlSugarInit.cs`

**主要特性：**
- ✅ 主从分离（Master-Slave 读写分离）
- ✅ 多数据库连接配置
- ✅ AOP 拦截（SQL 执行前/后/错误）
- ✅ 自动事务管理
- ✅ 删除/更新强制 WHERE 条件检查

**配置示例：**
```csharp
services.AddScoped<ISqlSugarClient>(o => {
    var client = new SqlSugarScope(configList);
    
    // SQL 执行前事件
    client.Aop.OnLogExecuting = (sql, pars) => {
        LogHelper.MysqlInfo(sql);
    };
    
    // SQL 执行后事件
    client.Aop.OnLogExecuted = (sql, pars) => {
        LogHelper.MysqlInfo($"耗时：{client.Ado.SqlExecutionTime.TotalMilliseconds}ms");
    };
    
    // 错误事件
    client.Aop.OnError = (exp) => {
        LogHelper.MysqlError(exp.Message);
    };
    
    return client;
});
```

### 技术选型对比

| 领域 | 技术选型 | 优势 |
|------|----------|------|
| API 网关 | Ocelot | 轻量、灵活、.NET 原生 |
| 服务发现 | Consul | 简单、内置健康检查 |
| 依赖注入 | Autofac | 功能强大、支持 AOP |
| ORM | SqlSugar | 国产、易用、功能全 |
| 缓存 | Redis + Memory | 高性能、分布式支持 |
| 对象映射 | Mapster | 性能最优、API 简洁 |
| 分布式事务 | CAP | 最终一致性、易集成 |
| 文件存储 | MinIO | 开源 S3 兼容、自部署 |
| 认证 | JWT | 无状态、跨平台 |
| 日志 | Serilog | 结构化、高性能 |
