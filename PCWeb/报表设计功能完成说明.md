## 🎯 使用指南

### 创建报表

1. **打开设计器**
   - 点击菜单"报表管理" → "报表设计器"

2. **配置基本信息**
   - 输入报表名称
   - 选择报表分类
   - 选择数据源

3. **编写 SQL 语句**
   ```sql
   SELECT 
     id,
     order_no,
     customer_name,
     amount,
     status,
     create_date
   FROM orders
   WHERE 1=1
   ```

4. **解析字段**
   - 点击"解析字段"按钮
   - 自动获取 SQL 中的字段

5. **配置字段**
   - 设置显示名称
   - 选择字段类型
   - 设置列宽和对齐
   - 勾选冻结/汇总

6. **配置查询条件**
   - 添加查询条件
   - 选择控件类型
   - 设置查询方式

7. **配置钻取链接**（可选）
   - 设置触发字段
   - 选择链接类型
   - 配置参数映射

8. **保存报表**
   - 点击"保存"按钮

9. **预览报表**
   - 点击"预览"按钮
   - 查看实际效果
